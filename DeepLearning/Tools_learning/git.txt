初始化：
    git init

管理文件（文件夹）
    git add 文件名
    git add .

个人信息配置：用户名，邮箱
    git config --global user.emial "xxx.@gmial.com"
    git config --global user.name "lutos"

生成版本：
    git commit -m "描述信息"

查看版本记录：
    git log

回滚版本：
    git log
    git reset --hard 版本号

前进版本：
    git reflog
    git reset --hard 版本号

查看分支：
    git branch

新建分之：
    git branch 分支名称

切换分支：
    git checkout 分支名称

合并分支：
    git merge 要合并的分支    # 需要先切换分支再合并

删除分支：
    git branch -d 分支名称

添加远程仓库：
    git remote add origin http://xxx

推送代码：
    git push -u origin 分支

克隆远程仓库代码：
    git clone http://xxx

切换分支：
    git checkout 分支名称

保持代码提交整洁：
    git rebase 分支

记录图形展示：
    git log --graph --pretty=format:"%h, %s"

git pull origin dev:
    等同于： git fetch origin dev + git merge dev

不产生分叉的pull方法：
    git fetch origin dev
    git rebase origin/dev

自动解决冲突软件：
    beyond compare
